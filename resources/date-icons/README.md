# 日期图标文件

这个目录包含预生成的日期图标文件。

## 文件命名规则

- `date-DD.svg` - 常规尺寸图标 (18x18px)
- `date-DD@2x.svg` - 视网膜屏幕图标 (36x36px)

其中：

- DD: 日期 (01-31)

## 文件数量

- 总共 62 个图标文件
- 31天 × 2种尺寸 = 62个文件

## 使用说明

这些图标由 `scripts/generateDateIcons.mjs` 脚本自动生成。
要重新生成图标，请运行：

```bash
npm run generate-icons
```

## 图标设计

- 简洁的日历图标设计
- 顶部有装订环效果
- 中央显示日期数字
- 使用系统字体确保清晰度
- 适配菜单栏尺寸 (18px) 和视网膜屏幕 (36px)
