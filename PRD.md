## 1. 产品概述

### 1.1 产品定位
万年历是一款轻量级桌面日历应用，专注于提供准确的中国农历、节假日信息，以及便捷的日期查询功能。采用系统托盘常驻方式，提供随时可唤出的快速查看体验。

### 1.2 目标用户
- 需要频繁查看农历/节假日的上班族
- 关注传统节气、黄历信息的用户
- 追求简洁、轻量工具的用户

### 1.3 核心价值主张
- **轻量快速**: 基于 Tauri 构建，体积小(<10MB)、启动快(<1s)
- **精准信息**: 准确的农历、节假日、节气数据
- **即点即看**: 托盘图标一键唤起，无需等待
- **原生体验**: 系统级集成，符合平台设计规范

### 1.4 产品边界
| 范围内 | 范围外 |
|--------|--------|
| 桌面端日历查看 | 移动端应用 |
| 本地节假日/农历数据 | 日程管理/日历订阅 |
| 系统托盘工具 | 数据导入导出/云同步 |
| 只读信息展示 | 自定义事件/提醒通知 |

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 日历视图 (Calendar View)
**优先级**: P0

**功能描述**:
- 月视图展示，支持月份/年份快速切换
- 公历日期主体显示
- 农历日期辅助显示（小字在公历下方）
- 今日高亮标识
- 选中日期视觉反馈
- 节假日/节气标识（小红点/小图标）

**详细规格**:
```
视图模式:
- 月视图 (默认): 6行 x 7列网格，显示42天
- 周起始: 支持周一/周日开始设置

日期显示层级:
1. 公历日期 (主显示，16-18px)
2. 农历日期 (副显示，10-12px，灰色)
3. 节假日/节气标记 (底部小圆点或文字)

视觉状态:
- 今日: 蓝色圆环背景 + 加粗文字
- 选中: 蓝色实心背景 + 白色文字
- 非当前月: 灰色文字，透明度60%
- 周末: 日期文字红色
```

**交互行为**:
- 点击日期: 选中该日期，显示详细信息面板
- 滚轮: 切换月份
- 拖拽: 不支持

#### 2.1.2 农历与黄历 (Lunar Calendar)
**优先级**: P0

**功能描述**:
- 准确的农历日期转换（1900-2100年）
- 干支纪年显示（甲子、乙丑等）
- 生肖属相
- 二十四节气
- 农历节日（春节、元宵、端午、中秋等）
- 公历节日（元旦、劳动、国庆等）
- 传统黄历信息（宜/忌，可选）

**详细规格**:
```
农历显示内容:
- 农历月日: 正月、二月、腊月、初一、十五等
- 干支: 甲子年、丙寅月、戊辰日
- 生肖: 鼠、牛、虎等（年度）

节气:
- 24节气: 立春、雨水、惊蛰...大寒
- 显示方式: 在对应日期下方标注节气名

节日分类:
- 法定节假日: 红色标识，显示放假/调休状态
- 传统节日: 橙色标识
- 公历节日: 蓝色标识
- 国际节日: 灰色标识

宜忌信息（可选功能）:
- 宜: 嫁娶、出行、开市等
- 忌: 动土、安葬等
- 吉神凶煞（可选简化版）
```

**数据精度**:
- 农历计算误差: <1秒
- 节气计算: 基于天文算法，精确到分钟
- 节假日数据: 支持国务院放假安排更新

#### 2.1.3 节假日展示 (Holiday Display)
**优先级**: P1

**功能描述**:
- 法定节假日自动标注
- 调休上班日特殊标识（如周日上班）
- 底部栏显示距离下一个假期的天数
- 节假日数据在线更新（年度更新）

**详细规格**:
```
法定节假日（中国）:
- 元旦: 1月1日
- 春节: 农历正月初一-初三
- 清明: 4月4日-6日（浮动）
- 劳动节: 5月1日-5日
- 端午: 农历五月初五
- 中秋: 农历八月十五
- 国庆: 10月1日-7日

调休标识:
- 周末上班（调休补班）: 日期右上角显示"班"字（红色），按周一-周五样式显示（非周末红字）
- 法定假日（含假期内周末）: 日期右上角显示"休"字（绿色）；日期文字红色

假期倒计时:
- 底部栏简要显示: "距离清明假期还有 23 天"
- 仅显示最近的下一个法定假期
```

**法定假日数据源与缓存**（v1.0.3 新增）:
```
数据源:
- 在线: holidays.js（GitHub 项目 chinese-lunar-calendar-for-mac 的 WanNianLi/Resources/vendors/holidays.js）
- Raw URL: https://raw.githubusercontent.com/zfdang/chinese-lunar-calendar-for-mac/master/WanNianLi/WanNianLi/Resources/vendors/holidays.js
- 文件格式: 注释含国务院公告摘要；变量 HOLIDAYADJUSTMENT 键值 "YYYYMMDD": "+"|"-"（+ 放假，- 调休上班）

解析规则:
- 从 HOLIDAYADJUSTMENT 提取该年所有日期的 +/- 标记
- 从注释中解析各假期“起止范围”（如 X月Y日至U月V日），用于将假期内的周末也标为法定假日并显示“休”

缓存策略:
- 按年份缓存至本地（应用数据目录），永不过期
- 无网络或拉取失败时：优先使用本地已有缓存；无缓存则降级为仅周末红字、无调休/法定角标

年份范围（以数据源为准）:
- 数据源目前仅包含 2020 年及以后的假日信息；仅对数据源内存在的年份做法定假日/调休处理
- 更早年份或数据源尚未覆盖的之后年份：不处理法定假日与调休，按普通周末/工作日显示（周末红字、无休/班角标）

展示与数据模型:
- 日期类型 holidayKind: statutory（法定假日）| weekend（普通周末）| workday（工作日/调休上班）
- 红色字体: 所有放假日（statutory 或 weekend）
- 右上角“休”: 仅 statutory；“班”: 仅 workday 且为调休补班日
```

### 3.2 视觉设计规范

**颜色系统**:
```
主色调:
- Primary: #3B82F6 (蓝色) - 选中、按钮、强调
- Secondary: #64748B (灰色) - 次要文字
- Success: #10B981 (绿色) - 假期、成功
- Warning: #F59E0B (橙色) - 节日、警告
- Danger: #EF4444 (红色) - 周末、上班日
- Background: rgba(255, 255, 255, 0.85) - 背景

文字颜色:
- Text Primary: #1E293B (主文字)
- Text Secondary: #64748B (次要文字)
- Text Disabled: #94A3B8 (禁用/非本月)

暗黑模式:
- Background: rgba(30, 41, 59, 0.9)
- Text: #F1F5F9 / #CBD5E1
```

**字体规范**:
```
字体栈:
- Windows: "Segoe UI", "Microsoft YaHei", sans-serif
- macOS: -apple-system, "PingFang SC", sans-serif
- Linux: "Noto Sans CJK SC", sans-serif

字号:
- 月份标题: 18px, bold
- 公历日期: 14px, medium
- 农历日期: 11px, regular
- 节气/节日: 10px, regular
- UI文字: 12-14px
```

**间距系统**:
```
- 窗口内边距: 16px
- 日历网格间距: 4px
- 单元格内边距: 4px

**效果**:
```
- 毛玻璃: backdrop-filter: blur(24px)

- 圆角: 窗口 16px, 单元格 8px, 按钮 6px
- 边框: 1px solid rgba(255, 255, 255, 0.5)
```

### 3.3 动画与过渡

```
- 月份切换: 300ms ease-out, 淡入淡出+轻微位移
- 日期选中: 150ms ease, 缩放+颜色变化
- 窗口显示: 200ms ease-out, 淡入+从托盘位置弹出

## 4. 技术需求

### 4.1 技术栈
- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite 6
- **样式**: Tailwind CSS 4
- **后端**: Rust + Tauri 2
- **状态管理**: React Context (轻量) / Zustand (复杂状态)
- **日期处理**: chrono (Rust), date-fns/dayjs (JS)

### 4.2 性能指标
- 启动时间: < 1秒（冷启动）
- 窗口唤出: < 100ms
- 月份切换: < 50ms
- 内存占用: < 50MB（典型使用）
- 包体积: < 10MB

### 4.3 兼容性
- Windows 10/11
- macOS 12+ ( Monterey 及以上)
- Linux (Ubuntu 20.04+, 其他发行版需测试)

### 4.4 数据存储
```
设置存储: 
- 方式: Tauri 配置 API / 本地 JSON
- 路径: %APPDATA%/wannianli/settings.json

节假日数据:
- 内置基础数据（1900-2050）
- 在线更新（年度更新）
```

### 4.5 第三方依赖
- **农历计算**: 内置查表算法（1900-2100 年数据）
- **日期处理**: chrono (Rust)
- **自动启动**: tauri-plugin-autostart
- **更新**: tauri-plugin-updater
